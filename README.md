# Cipher Encryption

Programming Language : Python <br>
GUI : PyQT

Installation
> conda env create -n <ENV_NAME> -f env.yml

Run Program
> python main.py

## Types of Encryption :
1. Vigenere Cipher 
   - Full Version
   - Auto-key Version
   - All ASCII Version (256 Characters)
2. Playfair Cipher
3. Affine Cipher
4. Hill Cipher
5. Enigma Cipher
6. Super Encryption
